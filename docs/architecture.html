Microsoft Azure Terraform Module v1.0

terraform-azure-monitor

Executive Summary

Comprehensive Terraform module for deploying Azure Monitor infrastructure including Log Analytics workspace, Application Insights, action groups (email/SMS/webhook/Logic App/Azure Function), metric alerts, log-based scheduled query alerts, activity log alerts, diagnostic settings for any Azure resource, and data collection rules for VM telemetry. Provides a complete observability platform for enterprise Azure environments.

Security Controls

Data Protection

- Configurable data retention (30-730 days)
- Daily ingestion quotas
- Internet ingestion/query toggle
- Sensitive output masking (keys, connection strings)

Alerting

- Service health alerts for outages
- Resource health monitoring
- Administrative activity tracking
- Auto-mitigation for alert recovery
- Common alert schema for standardization

Production Readiness Checklist

  • Log Analytics workspace with appropriate retention
  • Action groups configured for critical, warning, informational
  • Service health alerts active
  • Diagnostic settings for all critical resources
  • Configure metric alerts for CPU, memory, disk, latency
  • Set up log alerts for error patterns
  • Enable Application Insights for web applications
  • Deploy data collection rules for VMs
  • Configure daily quota limits for cost control
  • Test alert notification delivery
